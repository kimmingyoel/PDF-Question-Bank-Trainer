# 프로젝트 개요 및 목적  
이 프로젝트는 PDF나 DOCX 형식의 학습 자료로부터 문제를 추출하거나, 텍스트 기반 학습 내용을 입력받아 문제를 자동 생성하여 문제 은행 형태로 관리하고 반복 학습할 수 있도록 돕는 애플리케이션이다.  

로컬 환경에서 실행되는 AI 모델을 활용해 외부 API 의존 없이 문제 생성이 가능하며, 사용자가 틀린 문제나 중요하다고 표시한 문제를 중심으로 학습할 수 있도록 설계되어 있다.

### 기술 스택
백엔드는 Python 기반의 FastAPI로 구성되어 있으며, 데이터베이스는 PostgreSQL을 사용한다.  
문제 생성 기능은 Ollama를 통해 로컬에서 실행되는 LLM(gemma 계열 모델)과 LangChain을 활용한다.  
프론트엔드는 React와 TypeScript로 구성되어 있고, Vite 기반 개발 환경과 React Query를 사용해 API 통신을 관리한다. UI는 Material UI 컴포넌트를 기반으로 한다.

### 설치 방법
Python 3.11 이상, Node.js, PostgreSQL이 설치되어 있어야 한다.
리포지토리를 클론한 후 backend 디렉터리에서 Python 의존성을 설치하고, frontend 디렉터리에서 npm install을 실행한다.  
backend/.env.example, frontend/.env.example 파일을 각각 복사해 .env 파일을 생성한 뒤 필요한 환경 변수(DB 연결 정보, API 주소 등)를 설정한다.  
AI 문제 생성을 사용할 경우 Ollama를 설치하고 지정된 모델을 미리 받아두어야 한다.  

### 실행 방법
PostgreSQL과 Ollama 서버가 실행 중인 상태에서 backend 디렉터리에서 FastAPI 서버를 실행한다.  
이후 frontend 디렉터리에서 개발 서버를 실행하면 브라우저를 통해 애플리케이션에 접근할 수 있다.  
백엔드는 기본적으로 8000번 포트, 프론트엔드는 5173번 포트를 사용한다.

### 제공 기능

- PDF 및 DOCX 파일 업로드를 통한 문제 자동 추출
- 텍스트 입력 기반 AI 문제 생성
- 문제 은행 조회 및 문제 유형별 필터링
- 퀴즈 모드 제공(문제 순서 및 선택지 랜덤화)
- 북마크 기능 및 자주 틀린 문제 중심 학습
- 문제 풀이 결과 기록 및 즉시 피드백 제공
